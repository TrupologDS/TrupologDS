## Featured Work

### LLM Reliability Lab

[llm-reliability-lab](https://github.com/TrupologDS/llm-reliability-lab) is a reproducible research-engineering lab for LLM post-training, behavior evaluation, regression detection and reliability reporting.

Run 001 includes:

- Qwen2.5-0.5B-Instruct baseline vs LoRA SFT adapter
- 3,000 UltraChat SFT examples
- dataset builder and dataset validation
- SFT formatting and collator preflight
- baseline evaluation
- direct PEFT adapter comparison
- regression report
- qualitative error analysis

Run 001 showed mixed behavior: format compliance and overall rule-based score improved on a 60-prompt project-local reliability suite, while strict instruction-following regressed on exact-output and length/count constraints. The report treats this as a debugging signal, not a benchmark claim.
